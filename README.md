## 🎯 面试建议

### 准备策略
1. **必须掌握的TOP 5题目**：两数之和、反转链表、有效括号、字符串反转、合并有序链表
2. **重点练习数据结构**：数组、链表、栈、队列、字符串
3. **基础算法思想**：双指针、递归、迭代、简单动态规划

### 面试技巧
1. **先说思路**：不要直接写代码，先和面试官沟通解题思路
2. **考虑边界**：空输入、单元素、极值情况
3. **分析复杂度**：时间复杂度和空间复杂度
4. **写出测试**：简单的测试用例验证
5. **Swift优先**：大部分iOS岗位更看重Swift实现
